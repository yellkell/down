# DOWN (original, 2019-era)

The first version of DOWN, preserved exactly as it was: a single A-Frame
page with hand-placed stars and copy-pasted slide blocks — a first VR
project, warts and all.

Serve this folder with any static server and open `index.html`:

```bash
npx --yes serve . -l 5174
```

The modern IWSDK remaster lives at the repository root.
